# vnesports - Back-end
VN eSports Web production code for back-end
# Hướng dẫn sử dụng
```javascript
git clone https://github.com/hellonewday/vnesports
```
```nodejs
npm install
```
```javascript
npm start
```

Nodejs server host at: http://localhost:8888
